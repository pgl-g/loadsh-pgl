
# let和var的区别
/**
 * 
 *  1. let是es6 var是es5
 *  2. let是没有预处理机制，var是有预处理机制（不管变量是声明在什么位置，都会提升）
 *  3. let不能重复声明
 *  4. var对于作用域来说，var没有作用域只是声明变量，没有块级作用域，let有块级作用域，不会被外界干扰
 */
